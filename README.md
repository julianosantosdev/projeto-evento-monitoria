# GoTogether 🚀

O **GoTogether** é um aplicativo desenvolvido para reunir, de forma prática e acessível, informações sobre eventos, palestras, excursões, pesquisas de projetos e monitorias voltados para instituições. 

A proposta central do projeto é centralizar essas oportunidades em um único ambiente digital, facilitando o acesso dos usuários, promovendo a organização e aumentando o engajamento da comunidade.

---

## 📌 Funcionalidades Principais

### 🔐 Login & Acesso Rápido
* **Acesso Direto:** Facilidade para visualizar o conteúdo rapidamente através do botão **"Ver Eventos"**, que direciona o usuário direto para a listagem.

### 🏠 Tela Inicial (Home)
* **Barra de Pesquisa:** Para encontrar eventos rapidamente.
* **Menu de Navegação:** Facilidade para navegar entre as seções do app.
* **Destaques:** Exibição dos principais eventos do momento em posição privilegiada.
* **Ações Rápidas:** Botões intuitivos para **"Criar Evento"** e **"Sair"** (Logout).

### 📅 Listagem de Eventos
* Exibição de uma lista organizada e fluida com todos os eventos disponíveis na plataforma.

### ➕ Criar Evento (Formulário Dinâmico)
Interface intuitiva para cadastrar e publicar novos eventos. O formulário se adapta dinamicamente com base na escolha do usuário:
* **Campos Padrão:**
  * **Tipo de evento:** Menu de seleção (Ex: Evento, Palestra, Excursão, Monitoria, Projeto).
  * **Data e Hora:** Definição exata do dia e horário do acontecimento.
  * **Local:** Informações sobre onde o evento será realizado.
  * **Descrição:** Espaço para detalhamento e informações adicionais.
* **Campos Condicionais (Dinâmicos):**
  * Se o tipo selecionado for **Palestra** ➡️ Exibe o campo adicional **Palestrante**.
  * Se o tipo selecionado for **Excursão** ➡️ Exibe o campo adicional **Valor**.# Projeto-Evento
